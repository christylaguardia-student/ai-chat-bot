A simple NodeJS chat bot for searching eBay products.

by Christy La Guardia

Web App Source Code Bot Web Service Source Code

Built With

  • Bulma - UI framework
  • Express - Web framework
  • socket.io - Real-time chat engine
  • Dialogflow - Chat bot platform and API
  • eBay Finding API - eBay inventory keyword search
  • icanhazdadjoke API - Because this is a funny bot

How It Works

Ask the chat bot about a product.

You can specify new or used condition.

It will search eBay's inventory and return a top match.

The search results include:

  1. Product title
  2. Current bid price
  3. Product URL

Sample Messages

    I want to buy VR goggles

    I need a new computer find used laptop

    I'm looking for a used chandelier

    search iphone do you have guitars

    tell me a joke

Find a new or used product on eBay
