Shopping Chat Bot

by Christy La Guardia

A simple chat bot build using the socket.io real-time engine, a custom Dialogflow AI agent & API and the Bulma CSS framework.

View my source code on GitHub.

Sample Messages:

I want to buy a T-Shirt.

Track my order.

I'm looking for a gift.

Cancel my order.

Check what's in my cart.

Do you have any special offers.
