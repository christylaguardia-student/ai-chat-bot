eBay Shopping Chat Bot

by Christy La Guardia

About

A simple JavaScript chat bot built with NodeJS, the ExpressJS web framework and the Socket.io real-time engine.

The AI chat functionality was configured with a custom DialogFlow agent and integrated into the web app with the DialogFlow API.

eBay product and product best deal searches are handled with the eBay Finding API, hosted on a separte cloud service.

The UI was designed on the Bulma CSS framework.
